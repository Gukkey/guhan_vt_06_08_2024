# URL Shortner

API for an URL shortner made in 2 days for Vidyayatan Technologies. 

# Things accomplised (and planned for) in 2 days

- [x] Ratelimiting
- [ ] Docker support
- [ ] Live link
- [ ] Analytics

# Stack Used

- Java (Language)
- SpringBoot (Backend Framework)
- JUnit, Mockito (For testing, mocking etc..)
- Bucket4j (Rate limiting)
- Postgres (Database)
